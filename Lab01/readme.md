# Discord
 Discord is a communicable platform designed for creating communities. It supports text chat, voice calls, and screen sharing. Popular among gamers, developers, and study groups, Discord lets users join or create servers with organized channels.ORiginally popular amongs gamers, Discord has grown into platform used by students, developers, creators, and professionls for collaboration, study groups, project discussions.


# Github
 Github is a cloud-based platform for version control and collaboration using Git. Developer use Github to store code, track changes, work in term, and manage software projects. It also supports issues, pull requests, documentation, and CI/Cd workflows.
